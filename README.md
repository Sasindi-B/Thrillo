Thrillo is a travel and experience booking platform that connects **travellers** with **business owners** offering unique stays, events, and activities.  
This repository contains the **frontend application**, built with modern web technologies and deployed on **Vercel**.

---

## üöÄ Tech Stack

- **React** (Vite)
- **JavaScript / TypeScript**
- **Tailwind CSS**
- **Axios** (API communication)
- **React Router**
- **Vercel** (Deployment)

---

## üéØ Features

### Traveller
- Browse registered businesses via **Hotspots**
- View business details (about, gallery, location, availability)
- Book places using calendar-based date selection
- Receive booking and payment notifications
- Secure payment flow after booking confirmation

### Business Owner
- Manage business profile
- Upload up to **6 business images**
- Add Google Maps location link
- Receive booking requests via notifications
- Accept or decline bookings
